# tastemaker

Nieuw HDEUs-project — in opzet; productcode volgt.

## Werkwijze

Dit repo volgt het conventies-pakket
[HDEUs/claudeconventions](https://github.com/HDEUs/claudeconventions).
Startpunt: lees `CLAUDE.md` en `.claude/rules/` voordat je (met Claude Code)
aan de slag gaat. De lint/guardrail-configs (ESLint, gitleaks, husky
pre-commit, tsconfig) staan hier al klaar; de bijbehorende
installatiestappen staan in `lint/SETUP.md` van het conventies-repo en
horen bij de eerste projectscaffold.
